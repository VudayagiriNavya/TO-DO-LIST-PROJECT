# TO-DO-LIST-PROJECT

## Description
A simple web-based To-Do List app using HTML, CSS, and JavaScript.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Responsive design

1. Clone the repository:

2. Open index.html in a browser.
